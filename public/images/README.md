# Images Folder Structure

This folder contains all images used in the Hindonix website.

## Folder Structure

```
images/
├── logo/
│   ├── hindonix-logo-full.svg      (Logo with company name)
│   ├── hindonix-logo-icon.svg      (Icon only logo)
│   └── hindonix-logo-white.svg     (White version for dark backgrounds)
│
├── products/
│   ├── knobs/
│   │   ├── brass-knob-1.jpg
│   │   ├── wooden-knob-1.jpg
│   │   ├── cotton-knob-1.jpg
│   │   └── metal-knob-1.jpg
│   │
│   ├── door-handles/
│   │   ├── door-handle-1.jpg
│   │   └── door-handle-2.jpg
│   │
│   └── pull-handles/
│       ├── pull-handle-1.jpg
│       ├── pull-handle-2.jpg
│       └── pull-handle-3.jpg
│
├── hero/
│   └── hero-background.jpg         (Optional hero section background)
│
└── testimonials/
    ├── client-1.jpg
    ├── client-2.jpg
    ├── client-3.jpg
    └── client-4.jpg

```

## Image Guidelines

### Logo Files

- **hindonix-logo-full.svg**: Full logo with company name (recommended: SVG format)
- **hindonix-logo-icon.svg**: Icon-only version for smaller spaces
- **hindonix-logo-white.svg**: White version for use on dark backgrounds

### Product Images

- **Format**: JPG or PNG
- **Size**: 800x600px minimum (4:3 aspect ratio recommended)
- **Quality**: High resolution, well-lit product photos
- **Background**: White or neutral background preferred

### Testimonial Images

- **Format**: JPG or PNG
- **Size**: 120x120px minimum (square)
- **Quality**: Professional headshots

## How to Use

Simply place your images in the appropriate folders using the naming convention shown above. The website will automatically use these images.

### Example Paths in Code:

- Logo: `/images/logo/hindonix-logo-full.svg`
- Product: `/images/products/knobs/brass-knob-1.jpg`
- Testimonial: `/images/testimonials/client-1.jpg`
